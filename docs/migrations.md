# Migrations

## English

Apiagex records schema-related history in the `schema_migrations` table.

The admin `Migrations` panel shows applied migration records, including:

- scope
- migration name
- applied timestamp

This is useful for backups, upgrades, and restore checks.

## Hindi

Apiagex schema-related history ko `schema_migrations` table me record karta hai.

Admin ka `Migrations` panel applied migration records dikhata hai, including:

- scope
- migration name
- applied timestamp

Ye backups, upgrades, aur restore checks ke kaam aata hai.
