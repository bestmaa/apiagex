# Local Database

## English

Local development should support a file database through SQLite. This lets a developer run Apiagex without installing PostgreSQL or MySQL.

The starter boot path creates content type, field, entry, and audit tables in the same SQLite file.

```txt
Database: SQLite
File: ./data/apiagex.db
```

## Hindi

Local development ke liye SQLite file database support hona chahiye. Isse developer bina PostgreSQL ya MySQL install kiye Apiagex run kar sakta hai.

Starter boot path same SQLite file me content type, field, entry, aur audit tables create karta hai.

```txt
Database: SQLite
File: ./data/apiagex.db
```
