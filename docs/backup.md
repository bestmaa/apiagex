# Backup

## English

The admin UI can export the CMS state as a JSON bundle and restore it later.

What is included:

- content types
- content fields
- entries and version history
- media metadata and file bytes
- webhooks and delivery history
- audit log records
- recorded schema migrations

Use the admin `Backups` panel to download a bundle or restore one.

## Hindi

Admin UI CMS state ko JSON bundle ke form me export karta hai aur baad me restore bhi kar sakta hai.

Isme ye sab included hota hai:

- content types
- content fields
- entries aur version history
- media metadata aur file bytes
- webhooks aur delivery history
- audit log records
- recorded schema migrations

Admin ke `Backups` panel se bundle download ya restore karo.
